Pizza King rendeléskezelő

Üdvözlünk a Pizza King világában!

Ez a webalkalmazás a Pizza King pizzéria rendeléseinek kezelésére készült. A felületen külön menüpontok alatt található meg a JavaScript CRUD, a React CRUD, SPA, Fetch API, Axios és OOJS oldalak.

Funkciók

1. Főoldal

Bemutatja az alkalmazást, tartalmazza a navigációt, a készítők adatait, és egységes belépési pontként szolgál a teljes projekt számára.

2. JavaScript CRUD

A rendelések kezelése JavaScript segítségével történik egy tömbben: új rendelés felvétele, listázás, szerkesztés és törlés.

3. React CRUD

Ugyanennek a CRUD működésnek a React alapú változata useState segítségével, korszerűbb komponensalapú megközelítésben.

4. SPA(Single Page Application)

A React keretrendszerrel egy olyan felületet hoztunk létre, ahol a pizza számláló és az árkalkulátor között az oldal újratöltése nélkül, azonnal válthat a felhasználó.

5. Fetch API

A böngésző beépített Fetch funkciójával oldottuk meg az aszinkron adatforgalmat, így a felület várakozás nélkül, a háttérben kommunikál a PHP szerverrel.

6. Axios

A rendelések kezeléséhez az Axios könyvtárat használtuk, amely automatikus JSON-kezelésével és átlátható hibakezelésével teszi egyszerűbbé és stabilabbá a backenddel való kommunikációt.

7. OOJS

Az alkalmazás logikáját osztályokba és objektumokba szerveztük, ami a moduláris felépítésnek köszönhetően sokkal átláthatóbb és könnyebben karbantartható kódot eredményezett.
